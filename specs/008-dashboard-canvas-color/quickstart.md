# Quickstart: Dashboard Canvas Color

This guide explains how to change the background color of your dashboard.

## 1. Changing the Background Color

1.  Navigate to the **Dashboard**.
2.  Tap **Edit Mode** in the top bar.
3.  Tap the **Dashboard Settings** icon (cog icon) in the top bar.
4.  In the dialog that appears, select a new color from the **Canvas Color** palette (e.g., Gray or Blue).
5.  Tap **Save**.
6.  The dashboard background will update immediately.

## 2. Returning to Default

- To revert to the system's standard background color, select the **White** or **Default** option in the Canvas Color settings.

## 3. Persistence

- Your canvas color is saved automatically. It will be maintained when you restart the application or switch between layouts.
