# Quickstart: Smooth Grid Snapping

This guide explains the improved interaction model for the HMI dashboard.

## 1. Smooth Dragging

- When you press and move a widget in **Edit Mode**, it will now follow your finger perfectly.
- You will see a translucent **"Ghost Box"** underneath the widget. This box indicates where the widget will land when you let go.
- The ghost box snaps to the 80dp grid in real-time.

## 2. Spring Snapping

- When you release a widget, it will no longer "jump" into place.
- It will perform a short, smooth **sliding animation** into the final grid position shown by the ghost box.
- This provides visual confirmation that the widget has been correctly aligned.

## 3. Resizing

- Resizing now behaves with the same smoothness.
- The resize handle allows you to expand the widget freely, while the ghost preview shows the increments of 80dp cells.
- Upon release, the widget will animate to the final size.
