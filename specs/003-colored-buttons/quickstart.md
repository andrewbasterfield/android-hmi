# Quickstart: Colored Buttons

This guide explains how to use the new color customization feature for HMI buttons.

## 1. Adding a Colored Button

1.  Open the HMI application.
2.  Switch to **Edit Mode** using the button in the top bar.
3.  Click **Add Widget**.
4.  Select **BUTTON** as the widget type.
5.  In the **Color Selection** row, choose a color (e.g., Green).
6.  Enter the **Tag Address** for the button.
7.  Click **Add**.
8.  The button will appear on the dashboard with the selected background color.

## 2. Automatic Contrast

- If you choose a dark color (e.g., Red or Black), the button text will automatically turn **White**.
- If you choose a light color (e.g., Yellow or White), the button text will automatically turn **Black**.
- This ensures maximum readability in industrial environments.

## 3. Configuration Persistence

- Your color choices are automatically saved to the device's storage.
- When you restart the application or switch back to Run Mode, all customized colors are maintained.
